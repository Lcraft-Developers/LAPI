Lcraft APIs are APIs for Minecraft and Java.



## Maven
### Lcraft API
```
<dependency>
  <groupId>de.lcraft.api</groupId>
  <artifactId>api</artifactId>
  <version>(current Version)</version>
</dependency>
```
### Lcraft API (beta)
```
<dependency>
  <groupId>de.lcraft.api</groupId>
  <artifactId>api-beta</artifactId>
  <version>(current Version)</version>
</dependency>
```
